# artistscard-test



### 기술 스택

프론트: React

백엔드: Node.js

배포: AWS S3 (프론트 배포 및 음원 파일 저장), EC2 (서버), RDS (데이터베이스)



### 사용방법

1. 회원 가입 후, 로그인하면 음원 데이터 작성 페이지로 들어가게 됩니다.
2. 음원 파일의 아티스트, 곡명, 앨범명을 입력한 후, 업로드 할 음원 파일을 고릅니다.
3. 음원 파일이 완전히 올라갈 때까지 기다린 후, 작성 버튼을 누르면 AWS S3에 저장됩니다.
4. 사이드바를 열어 음원 데이터 리스트 페이지로 들어가면 작성한 데이터를 볼 수 있습니다.
5. 텍스트바에 곡명을 입력하면 리스트에 있는 음원 데이터 중에 곡명과 일치하는 데이터가 나옵니다.
6. 각 음원 데이터 카드에 연필 아이콘을 누르면 음원 데이터를 편집할 수 있습니다.
